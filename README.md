# Project Management System (Laravel Example)

## Project Overview
This is a **demo project using Laravel 10** for managing projects.  
It allows investors and borrowers to track projects, edit them, upload images, and manage their status easily.

**Core Features:**
- Create a new project.
- Edit an existing project.
- Delete a project along with all associated images.
- Change project status through predefined stages (Review, Pre-Approval, Funding, Active, Completed).
- Upload a main image and additional gallery images for each project.
- Display projects with search, pagination, and AJAX support.
- Automatically calculate project funding percentage.

---

## Requirements
- PHP >= 8.1
- Laravel 10
- MySQL / MariaDB
- Composer
